# chip8-emulator
An implementation of the CHIP-8 emulator written in C. 

Compiles on MacOS. Depends on SDL2.

## Build
In order to build the emulator you should have [SDL2](https://www.libsdl.org/download-2.0.php) framework installed.

```bash
  $ make
```

## Run
```bash
  $ ./chip8 path/to/game
```
